이 프로젝트는 **SASRec (Self-Attentive Sequential Recommendation)** 모델의 PyTorch 구현체입니다.

### 2. model.py - SASRec 모델

#### PointWiseFeedForward (lines 5-19)
```python
class PointWiseFeedForward(torch.nn.Module):
    def __init__(self, hidden_units, dropout_rate):
        self.conv1 = torch.nn.Conv1d(hidden_units, hidden_units, kernel_size=1)
        self.dropout1 = torch.nn.Dropout(p=dropout_rate)
        self.relu = torch.nn.ReLU()
        self.conv2 = torch.nn.Conv1d(hidden_units, hidden_units, kernel_size=1)
        self.dropout2 = torch.nn.Dropout(p=dropout_rate)
```
- Transformer의 Feed-Forward 네트워크
- 1D Convolution으로 구현된 position-wise fully connected layer

#### SASRec 모델 (lines 25-123)
**구조**:
```python
class SASRec(torch.nn.Module):
    def __init__(self, user_num, item_num, args):
        # Embedding layers
        self.item_emb = torch.nn.Embedding(item_num+1, hidden_units, padding_idx=0)
        self.pos_emb = torch.nn.Embedding(maxlen+1, hidden_units, padding_idx=0)

        # Transformer blocks
        self.attention_layernorms = torch.nn.ModuleList()
        self.attention_layers = torch.nn.ModuleList()  # Multi-head Self-Attention
        self.forward_layernorms = torch.nn.ModuleList()
        self.forward_layers = torch.nn.ModuleList()     # Feed-Forward

        self.last_layernorm = torch.nn.LayerNorm(hidden_units, eps=1e-8)
```

**주요 메서드**:

1. **log2feats()** (lines 65-95): 시퀀스를 feature로 변환
```python
def log2feats(self, log_seqs):
    # Item embedding + Positional embedding
    seqs = self.item_emb(log_seqs) * sqrt(d_model)
    seqs += self.pos_emb(positions)

    # Causal mask (미래 아이템 보지 못하도록)
    attention_mask = ~torch.tril(torch.ones((tl, tl)))

    # Transformer blocks
    for i in range(num_blocks):
        # Multi-head Self-Attention
        mha_outputs = self.attention_layers[i](seqs, seqs, seqs,
                                               attn_mask=attention_mask)
        seqs = self.attention_layernorms[i](seqs + mha_outputs)

        # Feed-Forward
        seqs = self.forward_layernorms[i](seqs + self.forward_layers[i](seqs))

    return self.last_layernorm(seqs)
```

2. **forward()** (lines 97-109): 학습용 forward pass
```python
def forward(self, user_ids, log_seqs, pos_seqs, neg_seqs):
    log_feats = self.log2feats(log_seqs)  # (batch, maxlen, hidden_units)

    pos_embs = self.item_emb(pos_seqs)    # positive items
    neg_embs = self.item_emb(neg_seqs)    # negative items

    # Inner product로 logits 계산
    pos_logits = (log_feats * pos_embs).sum(dim=-1)
    neg_logits = (log_feats * neg_embs).sum(dim=-1)

    return pos_logits, neg_logits
```

3. **predict()** (lines 111-122): 추론용 메서드
```python
def predict(self, user_ids, log_seqs, item_indices):
    log_feats = self.log2feats(log_seqs)
    final_feat = log_feats[:, -1, :]  # 마지막 시점의 representation

    item_embs = self.item_emb(item_indices)
    logits = item_embs.matmul(final_feat.unsqueeze(-1)).squeeze(-1)

    return logits  # 각 아이템에 대한 점수
```

**모델 특징**:
- **Self-Attention**: 시퀀스 내 아이템 간 관계 학습
- **Causal Mask**: 미래 정보 누출 방지
- **Positional Embedding**: 아이템 순서 정보 반영
- **Residual Connection + LayerNorm**: 안정적인 학습

```python
def build_index(dataset_name):
    ui_mat = np.loadtxt('data/%s.txt' % dataset_name, dtype=np.int32)

    # user-to-item, item-to-user 인덱스 생성
```

#### data_partition() (lines 96-125)

```python
def data_partition(fname):
    # data/[fname].txt 파일 읽기
    f = open('data/%s.txt' % fname, 'r')
    for line in f:
        u, i = line.rstrip().split(' ')
        User[u].append(i)

    # 각 사용자 시퀀스를 train/valid/test로 분할
    for user in User:
        if len(User[user]) < 4:
            user_train[user] = User[user]
            user_valid[user] = []
            user_test[user] = []
        else:
            user_train[user] = User[user][:-2]  # 마지막 2개 제외
            user_valid[user] = [User[user][-2]]  # 마지막에서 두 번째
            user_test[user] = [User[user][-1]]   # 마지막

    return [user_train, user_valid, user_test, usernum, itemnum]
```

#### WarpSampler 클래스 (lines 69-92)
```python
class WarpSampler(object):
    def __init__(self, User, usernum, itemnum, batch_size=64, maxlen=10, n_workers=1):
        # 멀티프로세싱으로 배치 생성
        self.result_queue = Queue(maxsize=n_workers * 10)
        for i in range(n_workers):
            self.processors.append(
                Process(target=sample_function, args=(...)))
            self.processors[-1].start()

    def next_batch(self):
        return self.result_queue.get()
```
- **멀티프로세싱 기반 배치 샘플러**
- 각 사용자의 시퀀스에서 (user, seq, pos, neg) 튜플 생성
- pos: 다음에 올 실제 아이템
- neg: 랜덤 negative 샘플

#### sample_function() (lines 33-66)
```python
def sample_function(user_train, usernum, itemnum, batch_size, maxlen, result_queue, SEED):
    def sample(uid):
        seq = np.zeros([maxlen], dtype=np.int32)
        pos = np.zeros([maxlen], dtype=np.int32)
        neg = np.zeros([maxlen], dtype=np.int32)

        # 사용자 시퀀스를 역순으로 처리
        nxt = user_train[uid][-1]
        idx = maxlen - 1
        for i in reversed(user_train[uid][:-1]):
            seq[idx] = i           # 현재 아이템
            pos[idx] = nxt         # 다음 아이템 (positive)
            neg[idx] = random_neq(1, itemnum+1, ts)  # negative 샘플
            nxt = i
            idx -= 1
            if idx == -1: break

        return (uid, seq, pos, neg)
```

#### evaluate() & evaluate_valid() (lines 129-220)
```python
def evaluate(model, dataset, args):
    for u in users:
        # 시퀀스 구성: train + valid
        seq = [valid[u][0]] + reversed(train[u])

        # 평가 아이템: test[u][0] (정답) + 100개 랜덤 negative
        item_idx = [test[u][0]] + [100 random negatives]

        # 모델 예측
        predictions = -model.predict([u], [seq], item_idx)
        rank = predictions.argsort().argsort()[0]

        # NDCG@10, HR@10 계산
        if rank < 10:
            NDCG += 1 / log2(rank + 2)
            HT += 1

    return NDCG / valid_user, HT / valid_user
```
- **평가 방식**: 1개 정답 + 100개 랜덤 negative 중 순위 측정
- 상위 10개 안에 정답이 들어오는지 확인

## 핵심 알고리즘

### 1. Self-Attention Mechanism
- 각 아이템이 시퀀스 내 다른 아이템들과의 관계를 학습
- Causal mask로 미래 정보 차단

### 2. Positional Encoding
- 시퀀스 순서 정보를 반영하기 위한 학습 가능한 positional embedding

### 3. Negative Sampling
- 각 positive item마다 1개의 random negative item 샘플링
- BCE Loss로 positive는 1, negative는 0으로 학습

### 4. Evaluation
- 1개 정답 + 100개 random negative 중 순위 평가
- Top-10 내 정답 포함 여부로 성능 측정
